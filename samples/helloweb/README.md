# Hello Web Sample

The output of a program makes a web page.

## Setup

Install [Node.js](http://nodejs.org).

## Run

With the command line:
```
node server
```

The output:
```
Server started, listening at port 8000
```

Then launch http://localhost:8000 in your browser. It works!
